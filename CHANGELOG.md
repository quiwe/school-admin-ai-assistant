# 更新日志

## 0.4.0 - 2026-05-11

开发者：maoqiu

- 软件图标外部白色背景改为透明背景，并同步到桌面程序、安装包和前端 favicon。
- 增加远程最低版本策略，未来可通过 `version-policy.json` 控制旧版本强制更新。
- FAQ 新增相似问题去重保护，导入时自动跳过重复项，手动新增时提示确认。
- 大模型设置新增 Claude、Gemini、腾讯混元、Mistral、Cohere、华为云盘古等 Provider 预设。
- 系统设置新增模型连接测试，保存配置前可验证 API Key、Base URL 和模型名。
- 系统设置新增数据备份与恢复，支持导出/导入 FAQ、知识库解析文本、历史记录和非密钥设置。
- 新增 macOS 在线打包流程，可生成 macOS `.app` 压缩包并上传到 GitHub Release。

## 0.3.0 - 2026-05-10

开发者：maoqiu

- 软件图标替换为新的蓝色气泡学士帽图标，并同步到桌面程序、安装包和前端 favicon。
- 回复工作台点击“保存 FAQ”后，会将当前问题和回复保存到 FAQ 管理列表。
- FAQ 管理新增 Excel 导入和导出功能，导入格式与知识库管理中的 Excel 导入 FAQ 保持一致。
- 历史记录优化“标记为常见问题”交互，保存后显示已保存状态。
- 历史记录新增单条删除、勾选、多选和批量删除功能。
- 优化回复工作台窗口模式和全屏模式布局，减少全屏两侧空白，并避免学生问题输入框过宽。

## 0.2.7 - 2026-05-09

开发者：maoqiu

- 修复回复链路：先检索 FAQ 和知识库，只有命中可靠依据时才调用大模型生成回复。
- FAQ 增加相似问法召回，学生问题不需要和 FAQ 完全一致也能命中标准答案。
- 强化中文知识库检索，改进论文规范等中文文档的命中率，并过滤弱相关依据。
- 大模型调用失败时在回复工作台明确显示 Provider、模型名和错误信息，不再用本地模板假装生成成功。
- 系统提示词进一步限制回答范围：只能基于知识库和 FAQ，不能用通用知识编造学校政策。
- 顶部文案改为桌面端语境，并新增启动检查 GitHub Release 更新、下载安装包和打开发布页功能。

## 0.2.6 - 2026-05-08

开发者：maoqiu

- 知识库上传增加老版 Office 格式支持：DOC、PPT、XLS。
- 增加 PPTX 解析，知识库支持 PDF、DOC、DOCX、PPT、PPTX、TXT、XLS、XLSX。
- Excel FAQ 导入同时支持 XLSX 和 XLS。
- 修复部分主按钮文字显示为空白的问题。
- 修复切换左侧页面后设置表单、知识库上传选择等临时内容丢失的问题。
- 旧版 DOC/PPT 采用 OLE 文本尽力提取，复杂文件建议另存为 DOCX、PPTX 或 PDF 后上传。

## 0.2.5 - 2026-05-08

开发者：maoqiu

- 设置页增加更醒目的“保存模型配置”按钮，并保存后显示当前 Provider 和模型名。
- 知识库管理上传按钮改为“保存文件到知识库”，上传后显示保存结果和 chunk 数。
- 知识库文件列表支持修改分类并单独保存分类。
- 桌面版强制将数据库和上传文件保存到软件目录下的 `data/`，避免数据写入错误位置。
- 修复安装器快捷方式图标路径，桌面和开始菜单快捷方式使用应用图标。
- 收窄 PyInstaller 打包依赖，避免本地无关 GUI 包影响打包。

## 0.2.4 - 2026-05-08

- Release 安装包会内置 WebView2 Evergreen Standalone 安装器，目标电脑未安装 WebView2 时自动静默安装。

## 0.2.3 - 2026-05-08

- 增加启动日志和本地错误提示页，便于定位打包环境中的启动异常。

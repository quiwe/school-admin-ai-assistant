# 更新日志

## 0.2.6 - 2026-05-08

开发者：maoqiu

- 知识库上传增加老版 Office 格式支持：DOC、PPT、XLS。
- 增加 PPTX 解析，知识库支持 PDF、DOC、DOCX、PPT、PPTX、TXT、XLS、XLSX。
- Excel FAQ 导入同时支持 XLSX 和 XLS。
- 修复部分主按钮文字显示为空白的问题。
- 修复切换左侧页面后设置表单、知识库上传选择等临时内容丢失的问题。
- 旧版 DOC/PPT 采用 OLE 文本尽力提取，复杂文件建议另存为 DOCX、PPTX 或 PDF 后上传。

## 0.2.5 - 2026-05-08

开发者：maoqiu

- 设置页增加更醒目的“保存模型配置”按钮，并保存后显示当前 Provider 和模型名。
- 知识库管理上传按钮改为“保存文件到知识库”，上传后显示保存结果和 chunk 数。
- 知识库文件列表支持修改分类并单独保存分类。
- 桌面版强制将数据库和上传文件保存到软件目录下的 `data/`，避免数据写入错误位置。
- 修复安装器快捷方式图标路径，桌面和开始菜单快捷方式使用院徽图标。
- 收窄 PyInstaller 打包依赖，避免本地无关 GUI 包影响打包。

## 0.2.4 - 2026-05-08

- 安装包增加 Microsoft Edge WebView2 Runtime 检测。
- Release 安装包会内置 WebView2 Evergreen Standalone 安装器，目标电脑未安装 WebView2 时自动静默安装。

## 0.2.3 - 2026-05-08

- 修复桌面版窗口打开后可能出现 `127.0.0.1 拒绝连接` 的问题。
- 后台服务改为直接加载 FastAPI 应用对象，并等待健康检查成功后再打开页面。
- 增加启动日志和本地错误提示页，便于定位打包环境中的启动异常。

## 0.2.2 - 2026-05-08

- 将中国科学院院徽图片设置为软件图标、安装包图标和前端 favicon。

## 0.2.1 - 2026-05-08

- 开发者信息更新为 maoqiu。

## 0.2.0 - 2026-05-08

- 改为 Windows 桌面版本，启动后打开原生桌面窗口，不再跳转默认浏览器。
- 设置页支持通过 API Key 识别模型列表，并选择或手动保存当前模型。
- Windows 安装包自动发布到 GitHub Release，并保留版本号、开发者和更新信息。

## 0.1.0 - 2026-05-07

- 完成高校行政 AI 回复助手 MVP。
- 支持回复工作台、知识库管理、FAQ 管理、历史记录和系统设置。
- 支持 PDF、DOCX、TXT、XLSX 文件解析和本地知识库检索。
- 支持常用大模型 Provider 配置和敏感问题人工核实。
